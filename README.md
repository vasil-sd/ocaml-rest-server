# Rest_server

It is a simple framework for developing RESTfull JSON-communicating web-applications.
Based on Http/af and Yojson.

## Installation

```bash
git clone https://github.com/vasil-sd/ocaml-rest-server.git
opam pin add ocaml-rest-server
```
